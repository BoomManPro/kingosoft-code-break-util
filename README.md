## 青果教务系统验证码识别

## 使用方式

1.通过maven引入依赖



```html
<dependency>
  <groupId>cn.boommanpro</groupId>
  <artifactId>kingosoft-code-break-util</artifactId>
  <version>1.0.0.RELEASE</version>
</dependency>
```



## 启动加速

因首次使用需要加载图片缓存,使用如下方法在项目启动可以加速初次使用
```java
Class.forName("TrainImageCache");
```
